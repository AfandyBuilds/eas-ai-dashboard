Imp. Guidelines :

All the Resources to start using Microsoft Copilot.
Developers to Integrate their IDE environment with Github Copilot Plugin to start using.
https://github.com/enterprises/ejadasa

Non-Developers (PM, BA, Architect, Consultant, Design, Tester etc)  start using Copilot Chat Environment. (Links attached)
https://m365.cloud.microsoft/chat

Resources need to stop using any personally subscribed or Public AI.
To get Copilot access, List of Resources at project level or department level need to raise IT ticket and ask for Manager Approval & Mohamed Al Banna Approval.
There are 2 types of access, and you need to mention the correct access type resource wise as follows :
For Developers : Github Business  :  Github Enterprise + Github Copilot Business
For Non-Developers : Github Business Only :  Github Copilot Business only, without repository access for Copilot Chat only users, in case they do not have by default access.
