Key Design Decisions

  • Two filter rows: Topic pills (top) + Source badges (below) — both independently toggleable
  • Featured card: Latest item gets a highlighted gradient border
  • Color-coded source badges: Each source has a unique color for instant recognition
  • Admin-only Refresh button: Top-right, only visible to admin role
  • "Last updated" timestamp: Always visible in the subtitle
  • Consistent with existing pages: Same dark theme, card style, and pill pattern as Skills Library
